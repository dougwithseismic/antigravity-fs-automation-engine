# Getting Started

Set up Antigravity locally to explore the hybrid execution model.

## Prerequisites

- **Node.js**: Version 20 or higher
- **pnpm**: Version 9 or higher
- **Docker**: For running Postgres

## Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-org/antigravity.git
    cd antigravity
    ```

2.  **Install dependencies**:
    ```bash
    pnpm install
    ```

3.  **Set up environment**:
    ```bash
    cp .env.example .env
    # Edit .env with your database credentials
    ```

4.  **Start the database**:
    ```bash
    docker-compose up -d db
    ```

## Running the Application

Start everything in development mode:

```bash
pnpm dev
```

This starts:
- **API Server**: [http://localhost:3002](http://localhost:3002) - Orchestration engine
- **Web App**: [http://localhost:3000](http://localhost:3000) - Visual workflow editor
- **Documentation**: [http://localhost:3001](http://localhost:3001) - What you're reading now

## Exploring the Component Library

To see all workflow nodes and UI components in Storybook:

```bash
cd packages/ui
pnpm storybook
```

This opens an interactive component explorer where you can:
- Browse all available workflow nodes
- Test components in isolation
- Understand the visual language
- See code examples

## First Workflow

1.  Open the **Web App** at [http://localhost:3000](http://localhost:3000)
2.  Create a new workflow
3.  Add nodes from the sidebar (try the "AI Agent Workflow" template)
4.  Connect nodes to define the flow
5.  Execute and watch it pause/resume for human input
