# Getting Started

Follow this guide to set up Antigravity on your local machine.

## Prerequisites

- **Node.js**: Version 18 or higher.
- **pnpm**: We use pnpm for package management.
- **Docker**: Required for running the local Supabase instance.

## Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-org/antigravity.git
    cd antigravity
    ```

2.  **Install dependencies**:
    ```bash
    pnpm install
    ```

3.  **Initialize Supabase**:
    ```bash
    pnpm exec supabase start
    ```

## Running the Application

Start the development server for all apps:

```bash
pnpm dev
```

This will start:
- **Web App**: [http://localhost:3000](http://localhost:3000)
- **Documentation**: [http://localhost:3001](http://localhost:3001)
- **API Server**: [http://localhost:3002](http://localhost:3002)

## First Steps

1.  Open the **Web App** at [http://localhost:3000](http://localhost:3000).
2.  Click **"Create Workflow"** to start a new automation canvas.
3.  Drag a **"Trigger"** node onto the canvas.
4.  Connect it to an **"Action"** node (e.g., `console.log`).
5.  Click **"Execute"** to run your first workflow in the browser!
